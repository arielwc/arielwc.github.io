Been Stuck on a Problem

Basically I was making a student gradebook, I had a list of names and a list of numbers. The numbers list was in rows of four, so one row for each student, each containing four numbers which were test scores, so each student had done four tests and the list listed the four test scores for each student. I made the blank gradebook and put the names in, that was fine. But I needed to put the numbers in and have the correct row of test scores align with the correct name. I got stuck on this for a while. I thought initially that I could use the same or a similar method to what I used to put the names in as the list of names and list of numbers were in the correct order. For example, the first row on the list of test scores was for Student One on the name list.
